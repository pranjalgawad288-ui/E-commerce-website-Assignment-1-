E-Commerce Website Assignment

Project Overview

This project is a modern E-Commerce Website developed using React.js. The website provides users with a seamless online shopping experience, allowing them to browse products, view product details, add items to the cart, and manage their shopping activities efficiently.

![image alt](https://github.com/pranjalgawad288-ui/E-commerce-website-Assignment-1-/blob/583e7253596ecb2cc6081fc8445bc3ad42c39739/Screenshot%202026-06-05%20154932.png)

Features

- Responsive and user-friendly interface
- Product listing and product cards
- Shopping cart functionality
- Add and remove products from the cart
- Dynamic cart updates
- Component-based architecture using React
- Fast and interactive user experience
- Integration with Supabase for backend services and data management

Technologies Used

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
- Supabase
- Vite

Project Structure

The project follows a modular component-based structure, making the code easy to maintain and scale. Components are organized into reusable files for better readability and development efficiency.
